# Rotas & Redes — Graphs (multi-project)

Este repositório contém quatro projetos de grafos:
- airports/: Rotas aéreas entre aeroportos (direcionado, ponderado)
- transport/: Rede de transporte público (estações e conexões)
- social/: Rede social (amigos)
- curriculum/: Disciplinas e pré-requisitos

Instruções:
- Python 3.8+
- Rode os exemplos das pastas ou execute os testes com: `pytest`

Estrutura de pastas e arquivos principais:
- airports/graph.py, airports/cli.py, airports/example_data.json
- transport/graph.py, transport/example_data.json
- social/graph.py, social/example_data.json
- curriculum/graph.py, curriculum/example_data.json
- tests/*.py
